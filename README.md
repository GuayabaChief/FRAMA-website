# FRAMA-website

Enfocado a mostrar productos con un caracter responsivo y agradable al usuario.

## Tecnologías usadas:
* Bootstrap 4.3.1
* JQuery
* Ajax

## Peculiaridades:
El sitio de momento se encuentra desplegado en el servicio Netlify con fines de prueba.
